# Advent of Code 2025
In TypeScript. I don't actually know the best way to set up a TS module, but this worked for me.

## Setup
1. `npm init` and relevant updates to package.json
2. `typescript tsc --init` and relevant updates to tsconfig.ts

## Usage
Add days with .ts modules logging answers, then run using e.g. `npx ts-node src/d01/d01.ts`
